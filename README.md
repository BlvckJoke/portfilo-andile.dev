## Getting started

```bash
git clone https://github.com/BlvckJoke/portfilo-andile.dev.git
cd portfilo-andile.dev
```

Then open `andile-portfolio.html` in your browser. No install needed.

## Deployment

Hosted free on [GitHub Pages](https://pages.github.com/) or [Netlify](https://netlify.com).

To deploy on GitHub Pages:
1. Go to your repo → Settings → Pages
2. Set source to `main` branch, `/ (root)`
3. Your site will be live at `https://blvckjoke.github.io/portfilo-andile.dev`
